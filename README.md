### Чат (Slack)
On-line приложения для общения в режиме реального времени!

### Установка для разработчиков
1. git clone git@github.com:michaeldobosh/frontend-project-12.git
2. make install
3. npm link
4. make start

### Системные требования
Версия node.js: 18.14.1 и выше
